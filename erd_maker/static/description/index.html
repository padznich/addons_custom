Description

Creates an Entity Relationship Diagram (ERD)

Automatic ERD

Generate an ERD with little to no effort.

Instructions
1. Activate developer mode
2. Go to 'Settings->Migration->Create ERD'
3. Select the Model and Transverse depth and click the 'Make ERD(html)' button

Diagram a module

Learn the database relationship of an installed module with ease

Instructions
1. Activate developer mode
2. Go to 'Settings->Migration->Create ERD (Module)'
3. Select the Module and click the 'Make ERD(html)' button
